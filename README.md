# CRUD NetBeans Simple Apps

Aplikasi **CRUD (Create, Read, Update, Delete)** sederhana berbasis **Java Desktop** yang dibuat menggunakan **NetBeans IDE**.

Project ini cocok untuk pembelajaran dasar:
- Java Swing
- Koneksi Database
- Manajemen Data CRUD

User Interface menggunakan dashboard dari:  
https://github.com/DJ-Raven/java-ui-dashboard-011
